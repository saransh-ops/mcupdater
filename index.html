About — MC Mod Updater

Description

MC Mod Updater is a browser-based tool that checks your Minecraft mods and resource packs for updates using the Modrinth API. No installation required — drop your files in, pick your MC version & loader, and it instantly shows what's up to date, what needs updating, and what's unavailable for your version.

How to use

1
Choose your Minecraft version, mod loader, and release channel in the sidebar.
2
Switch between Mods (.jar) and Resource Packs (.zip/.png) using the tabs.
3
Drop your files into the upload zone or click to browse. Check/uncheck what to include.
4
Click Check for Updates. Files are SHA-1 hashed locally — only the hash is sent to Modrinth.
5
Results show status badges. Use Download All or Needs Update Only to grab a ZIP.

Status & abbreviation meanings

✨ Update Available
Needs Update — a newer version exists for your MC version
✓ Up to date
Installed — your file matches the latest release
⚠ Older Version
Outdated — an older release of this mod is installed
MC Ver
Minecraft Version — e.g. 1.21.4, 1.20.1
SHA-1
Secure Hash Algorithm 1 — fingerprint used to identify your exact file
RP
Resource Pack — texture/sound override packs (.zip or .png)
dl
Downloads — total download count shown on Modrinth
Release
Stable release — fully tested, production-ready version
Beta
Beta — feature-complete but may have minor bugs
Alpha
Alpha — early/experimental, expect instability

📄 Terms of Service

Powered by Modrinth API

This tool uses the api.modrinth.com/v2 public API to identify mods and fetch version data. Usage is subject to Modrinth's Terms of Service.

⚡ Rate Limits

Modrinth enforces API rate limits. Checking too many mods in rapid succession may cause requests to be throttled or fail. Recommended: check fewer than 50 mods at a time and wait a moment between runs if you encounter errors.

🔒 Privacy & Data

All file hashing happens entirely in your browser. Your actual mod files are never uploaded — only the SHA-1 hash string is sent to Modrinth. No personal data is collected or stored by this tool.

📦 Downloads

Download links point directly to Modrinth's CDN. This tool does not host, redistribute, or modify any mod files.

⚠ Disclaimer

  • This tool is unofficial and not affiliated with Modrinth or Mojang.
  • Only mods listed on Modrinth can be identified — others show as "Not Found".
  • Always verify compatibility before installing any update.
